# Mushee Suite — Hedera Dashboard (Demo)

Sleek black + white, glassless dashboard for a Hedera-native demo.

## Quickstart
```bash
npm install
npm run dev
```

## Deploy
- Vercel: import the repo / upload the folder, build command: `npm run build`
